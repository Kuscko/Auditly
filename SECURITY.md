# Security Policy

Thank you for helping keep Auditly secure! Please follow these guidelines to report vulnerabilities or security issues.

## Supported Versions
- Only the latest release is supported for security updates.

## Reporting a Vulnerability
- Please do **not** open public issues for security problems.
- Email security reports to: [maintainer email or security@kuscko.com]
- Include a detailed description and steps to reproduce.
- We will acknowledge receipt within 3 business days and provide a timeline for resolution.

## Disclosure Policy
- We follow responsible disclosure and will coordinate with you before public disclosure.

## Security Best Practices
- Do not share secrets, credentials, or sensitive data in issues or pull requests.
- Review the [README.md](README.md) and [CONTRIBUTING.md](CONTRIBUTING.md) for secure usage and contribution guidelines.

---

For urgent issues, please use the email above. Thank you for helping keep Auditly safe!
